# morning-show

A small static fan-style site inspired by *The Morning Show*.

## Files

- `index.html` contains the page structure.
- `styles.css` contains the layout, colors, and typography.
- `script.js` contains the episode browser and local comment forum logic.

## Run locally

Open `index.html` directly in a browser for the simplest setup.

If you want to serve it over HTTP and have Node available, you can also use a
small static server of your choice.

## Notes

- The discussion forum is client-side only.
- Comments are stored in the browser with `localStorage`.
- There is no shared backend or user authentication in this version.
